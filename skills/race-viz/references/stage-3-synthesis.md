# Stage 3 — Synthesis

Requires the confirmed stage-2 record; merges the analysis outputs with the stage-1 research brief.

## Build

1. **Phase structure.** Name the acts. Tie every act boundary to a data divide — a zone entry, a lane split, a weather system arrival — not to narrative convenience. If an act boundary can't be pointed to on a chart, it isn't an act boundary.
2. **Per-benchmark loss decomposition.** One table per named reference boat, with visible arithmetic, and segments that sum to the official corrected delta — show the reconciliation line. Never publish a reference-free "we lost X here" (doctrine 5). Different benchmarks legitimately tell different stories; present them as such rather than averaging them into mush.
3. **Discrepancy register.** Every claim-in-circulation from stage 1 versus what the track shows: verdict confirmed / contradicted / partial / unresolvable, with an evidence pointer per row. This register is frequently the most differentiated output of the whole project — the track contradicting the circulating story is the insight nobody else has.
4. **Case studies**, ordered by lesson value rather than chronology.
5. **Lessons**, written in language that survives a team-facing cut — specific, non-blaming, actionable.

## Language rules

- Confidence-label every non-obvious claim: verified-fact / participant-recollection / testimony / inference. Keep the four categories distinct; a crew member's memory is not a track fact.
- Margins under `noise_floor_min` are framed as inside scoring noise — mandatory framing, not optional nuance (doctrine 6). The stage-1 controversy check determines how hard this framing must be.
- **Participant-supplied corrections are binding.** Record them, own them explicitly in the text (no silent fixes), and propagate each one to `events.yaml`, the dashboard data, and — when numeric — the regression assertions.

## Stage-3 stop — corrections record (hard stop)

Present the narrative spine and the full discrepancy register per the stop mechanic. Collect corrections — the record may say "none," but it must say so explicitly. Tier 1 may merge this review into the stage-2 stop.
